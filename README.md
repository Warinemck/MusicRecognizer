# Music Recognizer

Un widget de escritorio para Windows que reconoce la música que se está reproduciendo
en tu sistema y muestra la carátula en un vinilo animado, perfecto para el uso con obs.

Hecho con Python, Tkinter, y `shazamio`.

## Descargo de Responsabilidad

Este proyecto utiliza la biblioteca no oficial `shazamio` y puede
dejar de funcionar si la API de Shazam cambia.
